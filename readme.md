*Requires OS X 10.8+ or Linux. Windows support planned.*

## Dev

Built with [Electron](http://electron.atom.io).

###### Commands

- Init: `$ npm install`
- Run: `$ npm start`
- Build OS X: `$ npm run build-osx`
- Build Linux: `$ npm run build-linux`
- Build all: `$ npm run build` *(OS X only)*
